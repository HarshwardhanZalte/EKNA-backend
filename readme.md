# EKNA - (Enterprise Knowledge Navigation Assistant) - backend

## BUILDING
---
EKNA (Enterprise Knowledge Navigation Assistant) is an AI-powered document intelligence platform that helps individuals and organizations manage, search, summarize, and interact with their documents using natural language. It uses Retrieval-Augmented Generation (RAG), LangChain, Google Gemini, and pgvector to provide accurate, context-aware answers based on uploaded documents. The platform supports personal and organization workspaces, role-based access control, and is deployed on AWS using Django, React, PostgreSQL, and Docker.
